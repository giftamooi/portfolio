IoT Smoke Detection & Window Automation

An IoT system for detecting cigarette smoke and automatically controlling ventilation using ESP32 and MQTT.

A system built with two ESP32 boards communicating through MQTT: one detects cigarette smoke using an MQ-2 sensor, while the other responds automatically by controlling a servo motor for the window and a stepper motor for the fan.

Behind the scenes:

  • Implemented the automatic response subsystem (servo and stepper motor)
  • Set up MQTT communication between the two ESP32 boards
  • Built a real time monitoring dashboard using React
  • Assembled the hardware prototype
  • Wrote the project documentation

Invisible Watermarking (DCT)

Digital image watermarking using the DCT method, tested for robustness against JPEG compression.

A program that implements invisible watermarking on digital images using the Discrete Cosine Transform (DCT) method. The watermark is embedded into the Y channel (YCrCb color space) to keep the image visually unchanged, then tested for robustness against JPEG compression at different quality factors.

Behind the scenes:

  • Implemented invisible watermarking using DCT with the YCrCb color space
  • Evaluated watermark robustness against JPEG compression at five different quality factors
  • Analyzed extraction results using correlation values and determined the robustness threshold
