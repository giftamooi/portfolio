# Gifta's Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

🔗 **Live site:** https://giftamooi.github.io/portfolio/


## Features

- Interactive sliding panels with smooth animations
- Expandable project cards
- Show more/less toggle for activity lists
- Responsive layout


## CI/CD & Deployment

This project uses **GitHub Pages** for continuous deployment. Every time changes are pushed to the `main` branch, GitHub automatically rebuilds and republishes the site, no manual upload required.

**Workflow:**
1. Edit code locally
2. `git add .`
3. `git commit -m "your message"`
4. `git push`
5. GitHub Pages automatically builds and deploys the updated site to the live URL above
